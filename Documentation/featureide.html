Installtion

FeatureIDE comes in pre-packaged Eclipse versions for Windows that can be downloaded from its official website: http://wwwiti.cs.uni-magdeburg.de/iti_db/research/featureide/#download. Our algorithm was implemented in v3.1.0. The current version is v3.3.0. Currently, there exist only a pre-packaged version for Windows. If you would like to use FeatureIDE with Linux or Mac OS, download the source code from https://github.com/FeatureIDE/FeatureIDE and import the plug-in projects into your Eclipse (needs EMF). Then start a new Eclipse instance from the core.fm package.
